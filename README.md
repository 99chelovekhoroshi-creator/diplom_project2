Parking Data System-Diploma Project

О проекте
Это мой дипломный проект-система, которая автоматически распознаёт госномера автомобилей с видео или изображений и сохраняет их в базу данных.
Идея в том, чтобы автоматизировать процесс учёта въезда и выезда машин на парковке университета.

Цель: сделать рабочий прототип, который: 
1)обрабатывает видео с камер;
2)распознаёт номера автомобилей;
3)сохраняет результаты в базу данных для дальнейшего анализа.

Используемые инструменты:

1)Python
2)OpenCV-для обработки изображений и видео
3)NumPy, Pandas — для работы с данными
4)SQLite — база данных, где хранятся распознанные номера
5)Дополнительно: os, time, imutils

Что внутри:
1)main.py, main2.py, 123.py, 456.py — основные скрипты для распознавания и записи данных
2)car_numbers.db — база данных с распознанными номерами
3)Папки с видео и изображениями (.MOV, .mp4, .png) — тестовые материалы
4)requirements.txt — список зависимостей проекта

Результат
Получился рабочий прототип, который распознаёт номера с видео и фото и сохраняет их в базу данных.
Проект можно доработать для использования в парковочных системах или автоматизированном контроле доступа.
